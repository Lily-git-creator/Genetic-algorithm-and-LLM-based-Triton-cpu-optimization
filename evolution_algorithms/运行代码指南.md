# 遗传算法代码运行指南

## 代码结构
- init_generator.py 程序初始化种群
- init_generator_llm.py LLM初始化种群
- main.py  主函数
  - evaluator.py 评估函数
  - evoluter.py  进化函数
  - llm_handler.py 模型调用接口

- run_benchmark.sh 测试脚本
- visualize_result.py 可视化分析


## 如何运行
1. 复现测试结果：
```bash
bash run_benchmark.sh
```

2. 运行爬山算法
```bash
python main.py --mode para --budget 10 --pop_size 3
```
3. 运行遗传算法
```bash
python main.py --mode ga --budget 10 --pop_size 3
```

4. 运行差分遗传算法
```bash
python main.py --mode ga --budget 10 --pop_size 3
```

## 参数配置
1. --mode :
   1. para:爬山算法
   2. ga:进化算法
   3. de:差分进化算法

2. --budget 迭代次数
3. --pop_size 种群初始数量
